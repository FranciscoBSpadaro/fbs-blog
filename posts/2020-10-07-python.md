### Solução do problema:

```python
produtos = []
quantidade = []
valor = []
resultado = []


for cont in range(0,3):
    produtos.append(str(input('Digite o produto:')))
    quantidade.append(int(input('Digite a quantidade:')))
    valor.append(float(input('Digite o Valor , não utilize virgula:')))
    
for x,y in enumerate(valor):
     resultado.append(y*quantidade[x])


print(f'O Produto:{produtos[0]} foi Vendido: {quantidade[0]} vezes ,  pelo valor de  R$ {valor[0]}  e o valor total foi de : R$ {resultado[0]} ')
print(f'O Produto:{produtos[1]} foi Vendido: {quantidade[1]} vezes ,  pelo valor de  R$ {valor[1]}  e o valor total foi de : R$ {resultado[1]} ')
print(f'O Produto:{produtos[2]} foi Vendido: {quantidade[2]} vezes ,  pelo valor de  R$ {valor[2]}  e o valor total foi de : R$ {resultado[2]} ')
print(f'O Valor Total de todas as Vendas é de  R$ {(sum(resultado))} ' )
```

### **Sequência de Fibonacci**

![](/assets/img/leofibo.png)

https://pt.wikipedia.org/wiki/Sequ%C3%AAncia_de_Fibonacci

**Código Python Para Calcular a Sequência de Fibonacci**

```python
print('-'*33)
print('Sequência de Fibonacci')
print('-'*33)

termos = int(input('Digite Quantas Sequência de termos Fibonacci você quer exibir : '))

termo0 = 0
termo1 = 1

print('-'*33)
print(f'{termo0} → {termo1}', end="")

contador = 3
while contador <= termos:
    termox = termo0 + termo1
    print('→ {} '.format(termox), end="")
    termo0 = termo1
    termo1 = termox
    contador += 1
    
    
print('→ Fim da Sequência')
```

Output :

![](/assets/img/outputfibonacci.png)

Comentários :

Antes de elaborar o código foi preciso entender a logica da sequencia de Fibonacci , Na [](https://pt.wikipedia.org/wiki/Matem%C3%A1tica "Matemática")matemática, a  **Sequência de Fibonacci** é uma [](https://pt.wikipedia.org/wiki/Sequ%C3%AAncia_de_inteiros "Sequência de inteiros")sequência de números inteiros , começando normalmente por 0 e 1, na qual, cada termo subsequente corresponde à [](https://pt.wikipedia.org/wiki/Soma_(aritm%C3%A9tica) "Soma (aritmética)")soma dos dois anteriores.

Quando o código é iniciado o Python executa as primeiras linhas do código onde o print exibe o valor de termo0 e termo1 onde estão com o valor declarado de 0 e 1 

porem quando o código chega na estrutura de repetição while o código só vai rodar oque está dentro da estrutura do escopo do while que vai dar um looping de repetição pela quantidade definida pela variável ' termos ' onde o usuário ira digitar quantos termos devem ser exibidos na tela , então o código não vai ser executado desde as primeiras linhas e sim a sequencia vai sendo impressa na tela a cada looping de repetição do while , onde o valor do termo 0 passar a ser o valor do termo1 e o termo1 passa a ter o valor do termox que é a soma de termo0 e termo1 e assim por diante.

**Contador = 3** , o contador foi definido a partir do 3 pois a soma  já foi iniciada quando foi definido o valor do termo 0 e 1 e o termox .

e a linha de código ' **print('→ {} '.format(termox), end="")** '  fica no meio da estrutura do while pois  é dessa forma que o print ira imprimir um termo a cada sequencia de looping , pois se fosse colocado no final do código ele só iria colocar o valor total de todas as sequencias e o parâmetro ' end="" , faz com que cada termo seja impresso na mesma linha em sequência.

**Utilizando Função Recursiva :**

**A forma mais elegante de se elaborar um código para a sequencia de Fibonacci ou para um Fatorial o mais comum é utilizar a função recursiva como no exemplo abaixo :**

```python
def fibonacci(i, b=0, c=1):
    if i != 0:
        print(b, end=' -> ')
        b = b + c
        c = b - c
        fibonacci(i-1, b=b, c=c)
        return 'FIM'


print(fibonacci(int(input('Quantos termos vc quer mostrar: '))))
```

**Comentários :** 

**podemos observar que o código agora possui menos linhas  , porem a forma que o código vai ser executado pelo Python vai ser diferente , a função recursiva usa um método que podemos chamar de call stack onde ele salva uma pilha de chamadas na memoria toda vez que a função chamar ela mesma por isso o nome recursiva pois a função chama ela mesma o tempo todo ate o ponto de parada , o ponto de parada vai ser definido no if , onde i é o numero inteiro que foi solicitado pelo input para o usuário .**

**no parametro fibonacci(i-1, b=b, c=c)  onde i-1 significa que a cada ciclo de execução será subtraído o valor do numero inteiro  até chegar a 0 onde é o ponto de parada e b=b e c=c atualiza o valor dentro da def fibonacci(i, b=0 , c=1 ) da mesma forma que a cada ciclo o valor i vai ser subtraído , a cada ciclo o valor de b e c será alterado . e o print vai imprimir a cada ciclo o valor correspondente de b já o c vai ser um valor negativo que não será exibido**

### **Fatorial :**

**Fatorial é um número natural inteiro positivo, o qual é representado por n! O fatorial de um número é calculado pela multiplicação desse número por todos os seus antecessores até chegar ao número 1.**

**ex : 5x4x3x2x1 = 120**

**então seguindo o exemplo a cima vamos ao código primeiro usando o While:**

```python
contador = int(input(' Digite um número para calcular o Fatorial'))
fatorial = 1

print(f'Fatorial de {contador}! ', end="")

while contador > 0:
    print(f'{contador}', end="")
    print('x' if contador > 1 else '=' , end="")
    fatorial *= contador
    contador -= 1
print(f'{fatorial}')    
```
